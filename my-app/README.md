# 뺏지뺏지

### 상세 설명
2023년 고려대학교 소프트웨어 창업학회 NEXT에서 진행된 2W1W(2Weeks 1Web) 스터디 두번째 도전으로, 축제 기간 중 대기 및 공연 시간에 주위 인원과 편하게 즐길 수 있는 게임입니다.


### 개발 기간

2023.09.02 ~ 2023.09.08


### 주요 스택
React, Socket.io
